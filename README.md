# footwear-defect-inspection
Detects common footwear defects like stitching and knitting errors
Hướng dẫn cài đặt
Mở Command Prompt:

1. Clone repository

        git clone <repo_url>
  
        cd fe_footwear-defect-inspection
  
        cd frontend

2. Install library in reactjs project 

        npm install

3. Start the project 

        npm start 

  Truy cập tại: http://127.0.0.1:3000/login
